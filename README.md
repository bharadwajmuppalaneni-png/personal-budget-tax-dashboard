# Personal Budget & Tax Optimization Dashboard (India)

## Objective
To analyze monthly income, expenses, and savings of a salaried individual and identify opportunities for better financial planning and tax optimization.

## Tools Used
- Microsoft Excel
- Basic Financial Analysis

## Key Features
- Monthly expense tracking
- Savings calculation
- Tax estimation (basic slab system)
- Visual dashboard for insights

## Key Insights
- Identified high spending categories (Rent and Food)
- Savings rate fluctuates between 30%-50%
- Tax liability can be reduced with better planning

## Business Impact
This model can help individuals:
- Improve budgeting discipline
- Increase savings rate
- Plan taxes more efficiently

## Files Included
- budget_data.xlsx (dataset + calculations)
- expense breakdownpie chart (visual insights)
- monthly expense bar chart (visual insights)
- Savings trend line chart (visual insights)
